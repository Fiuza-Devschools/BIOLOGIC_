# 🌱 BIOLOGIC

O **BIOLOGIC** é um projeto desenvolvido para feiras de tecnologia e inovação, com o objetivo de auxiliar no monitoramento e manejo de pequenas hortas por meio da automação e do uso de sensores.

O sistema busca promover práticas de agricultura sustentável, oferecendo ao usuário informações sobre as condições da horta e auxiliando na tomada de decisões.

## 📌 Objetivos

* Monitorar as condições da horta em tempo real.
* Auxiliar o usuário na irrigação e nos cuidados com as plantas.
* Incentivar o uso consciente dos recursos naturais.
* Demonstrar a aplicação da tecnologia na agricultura.

## ⚙️ Funcionalidades

* Monitoramento da umidade do solo.
* Monitoramento da temperatura.
* Monitoramento da luminosidade.
* Monitoramento do nível de água.
* Monitoramento do pH da água.
* Dashboard para visualização das informações.
* Recomendações ao usuário com base nos dados coletados.

## 🛠️ Tecnologias Utilizadas

### Software

* HTML5
* CSS3
* JavaScript

### Hardware

* Arduino UNO
* Sensor de Umidade do Solo
* Sensor de Temperatura
* Sensor de Luminosidade (LDR)
* Sensor de Nível de Água
* Sensor de pH
* Módulo Relé
